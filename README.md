# YOLO-based-Air-to-Air-UAV-Detection
Pre-published version of research on vision-based air-to-air UAV detection using YOLO architectures, focusing on small object detection and real-time performance.

# About the Work
This study presents a comprehensive analysis of vision-based air-to-air UAV detection using various YOLO-based deep learning architectures. The work focuses on improving detection performance for aerial scenarios, particularly addressing challenges such as small object detection, complex backgrounds, and real-time processing constraints.
